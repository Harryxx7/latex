## 🚀 英语专区长期 Pair Programming 更新机制

在 **`English Reading`** 专区中，我们建立了一套极度无痛、高效的“日常积累 ➔ AI 自动重构”工作流：

1.  **日常积累**：当您在刷真题或复习时积累了新的心得，直接在大白话积累本 **[append_notes.md](file:///Users/harry/Desktop/latex/English%20Reading/append_notes.md)** 底部进行无脑追加（里面已为您备好“长难句”和“方法论”的极简套用模板）。
2.  **AI 一键排版与编译**：积累一定量后，直接对 AI 说：
    > *“请把我 `append_notes.md` 中新追加的内容整理到 `revision_handbook.tex` 手册中，并重新编译 PDF。”*
3.  **打印实体复习**：AI 将在后台自动完成所有的对齐表格、音标代码化以及 XeLaTeX 编译，您直接拉取最新的 **[revision_handbook.pdf](file:///Users/harry/Desktop/latex/English%20Reading/revision_handbook.pdf)** 打印出来双面背诵即可！
